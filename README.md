# Loan-Application-System

## Project Description

The Loan Application System is a web-based application developed to simplify and automate the loan application process. The system allows customers to register, log in, submit their personal and guarantor information, apply for loans, and track the status of their loan applications.

Administrators can manage loan requests, approve or reject applications, and monitor loan repayments through the admin dashboard

    ## objective
To allow customers to register and log into the system securely
To enable users to submit personal and guarantor information online.
To provide customers with the ability to apply for loans electronically.
To help administrators manage and review loan applications efficiently.
To allow administrators to approve or reject loan requests.
To monitor and track loan repayment records.
To improve accuracy, efficiency, and accessibility in loan management processes.

## Features

## Customer Features
User registration and login
Update personal information
Add guarantor details
Apply for a loan
View loan application status
Track repayment information

## Admin Features
Admin login dashboard
View all loan applications
Approve or reject loan requests
Manage customer records
Monitor loan repayments
